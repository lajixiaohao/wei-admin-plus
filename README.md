## wei-admin-plus

wei-admin-plus是wei-admin(Vue2.x)前端框架升级版，基于Vue3及其他最新技术栈打造。

#### 技术栈

`Vite`+`Vue3`+`TypeScript`+`Element plus`+`Pinia`+`Vue-Route4`

#### 项目克隆

```bash
git clone git@github.com:lajixiaohao/wei-admin-plus.git
```

#### 项目安装

```bash
npm i
```

#### 运行本地服务

```bash
npm run dev
```

#### 打包

```bash
npm run build
```
